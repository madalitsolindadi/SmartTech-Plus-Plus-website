About MySong BookApp

This is a SmartTech++ Smart eSongBook Library App.

Its operations include:

  1. Create a song book (with Title).
  2. Add a song book (equivalent to requesting that book).
  3. Add songs (with Title, Info & Body/lyrics) into a book (for Admin only).
  4. View songs on the screen display.
  5. Navigate to any desired song by clicking the title, or search for any text or number.
  6. Edit songs (for Admin only).
  7. Add members (other people who can access your song book) (for Admin only).
  8. View members.
  9. Remove members (for Admin only).
  10. Make member Admin or Dismiss as Admin (for Admin only).
and is able to export a systematic PDF song book document.

The app is powered by the secure and reliable Google Firebase Database Management System for its authentication, data storage and organisation.

You only need to have an email address for you to access the App.
